## Introduction

This command can create Table of content for gitlab with good markdown.

## Installation

```shell
sudo mkdir -p /opt
cd /opt
sudo git clone https://github.com/kwort/gltm gltm
sudo ln -s /opt/gltm/gltm.php /usr/bin/gltm
```

## Usage

```shell
gltm [FILE]
```
